# Stock Market Analysis Dashboard

## Project Overview

This project focuses on quantitative analysis of Indian IT sector stocks using Python and Power BI. Historical stock data was analyzed to evaluate performance, volatility, momentum, and risk-adjusted returns.

## Objective

To develop an interactive analytics dashboard that helps users understand stock behavior and make data-driven investment decisions.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* yFinance API
* Power BI

## Key Analysis Performed

* Daily Return Analysis
* Rolling Volatility Analysis
* Relative Strength Index (RSI)
* Correlation Analysis
* Sharpe Ratio Calculation
* Performance Comparison

## Dashboard Features

* Interactive stock performance visualization
* Volatility tracking
* RSI signal monitoring
* Correlation insights
* Risk-adjusted performance evaluation

## Project Outcome

Successfully transformed raw stock market data into actionable business insights through quantitative analysis and interactive visualization.
